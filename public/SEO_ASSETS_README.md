# SEO Assets Needed

## Required Images (Create these!)

### 1. OG Image (og-image.png)
- **Size:** 1200 x 630 pixels
- **Use:** Social media sharing (Facebook, LinkedIn, Twitter)
- **Design Tips:**
  - Include logo and tagline
  - "AI Mock Interview" text
  - Dark background (#0f172a) with blue accent (#3b82f6)
  - Show a mockup of the interview screen

### 2. App Icons
- **icon-192.png** - 192x192 pixels (PWA icon)
- **icon-512.png** - 512x512 pixels (PWA icon)
- **favicon.ico** - 32x32 pixels

### 3. Apple Touch Icon
- **apple-touch-icon.png** - 180x180 pixels

## Free Tools to Create:
- Canva (canva.com) - Free templates
- Figma (figma.com) - Design tool
- Remove.bg - Background removal

## After Creating:
1. Place all images in `/public/` folder
2. Rebuild and deploy

## Google Search Console Setup:
1. Go to: https://search.google.com/search-console
2. Add property: sunadhedutech.com
3. Verify ownership (HTML tag method)
4. Replace "ADD_YOUR_GOOGLE_VERIFICATION_CODE" in layout.tsx
5. Submit sitemap: https://sunadhedutech.com/sitemap.xml
